NEXORA - Sistema de Exibição de Produtos via CatalogX

Este projeto é um sistema web moderno para exibição de produtos, desenvolvido com foco em uma experiência de usuário otimizada e design responsivo.

Principais Características:

1. Layout Responsivo
   - Grid flexível que se adapta a diferentes tamanhos de tela
   - Cards de produtos otimizados para visualização em dispositivos móveis e desktop
   - Media queries para garantir a melhor experiência em qualquer dispositivo

2. Cards de Produtos
   - Design moderno com efeitos de hover
   - Exibição de imagens com zoom suave
   - Sistema de tags para destacar características especiais (Lançamento, Frete Grátis, etc.)
   - Avaliações com sistema de estrelas
   - Preços com suporte a parcelamento e destaque de economia

3. Recursos Visuais
   - Animações suaves para interações
   - Sistema de cores consistente
   - Tipografia otimizada para leitura
   - Ícones intuitivos

4. Organização do Código
   - CSS modularizado (style.css para estilos gerais, produtos.css para cards)
   - Estrutura semântica HTML
   - Sistema de grid flexível para produtos
   - Media queries organizadas por breakpoints

5. Performance
   - Carregamento otimizado de imagens
   - Animações suaves sem impacto na performance
   - Código CSS otimizado e organizado

O projeto utiliza uma estrutura de arquivos organizada, com separação clara entre estilos gerais e específicos de produtos, facilitando a manutenção e escalabilidade do código.
